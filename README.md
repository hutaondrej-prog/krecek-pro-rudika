# 🐹 Dobrodružství s křečkem pro Rudíka

Interaktivní webová hra pro 3leté děti s pěti zábavnými úkoly, které vedou k odhalení videa s křečkem.

## 📁 Soubory v projektu

- `index.html` - Hlavní HTML soubor
- `style.css` - Styly a animace
- `script.js` - Herní logika
- `Video_s_kr_ec_kem_pro_Rudi_ka.mp4` - Finální video s křečkem
- `README.md` - Tento soubor

## 🎮 Úkoly ve hře

1. **Nakrm křečka** - Drag & Drop jídla do mističky
2. **Najdi křečka v hoblinách** - Stírací los
3. **Poskládej kamaráda** - Jednoduché puzzle
4. **Chyť semínko** - Hra na postřeh
5. **Udělej křečkovi teplo** - Klikací hra s barvami

Po dokončení všech úkolů se objeví video s křečkem!

## 🚀 Jak nahrát na GitHub

### 1. Vytvoř nový repository na GitHubu
- Jdi na https://github.com/new
- Pojmenuj ho např. `krecek-pro-rudika`
- **DŮLEŽITÉ:** Zaškrtni "Add a README file"
- Klikni "Create repository"

### 2. Nahraj soubory
- V repozitáři klikni na "Add file" → "Upload files"
- Přetáhni sem všechny soubory:
  - index.html
  - style.css
  - script.js
  - Video_s_kr_ec_kem_pro_Rudi_ka.mp4
- Klikni "Commit changes"

### 3. Aktivuj GitHub Pages
- Jdi do "Settings" → "Pages"
- U "Source" vyber "main" branch
- Klikni "Save"
- **Za chvíli dostaneš URL**, kde bude hra dostupná (např. `https://tvoje-jmeno.github.io/krecek-pro-rudika/`)

## 📱 Použití

Hra je optimalizovaná pro dotykové obrazovky (mobily, tablety). 
Jednoduše otevři URL na mobilu/tabletu a Rudík může začít hrát!
