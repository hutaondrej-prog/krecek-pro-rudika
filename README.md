# 🐹 Dobrodružství s křečkem pro Rudíka

Interaktivní webová hra pro 3leté děti se 4 zábavnými úkoly, které vedou k odhalení videa s křečkem.

## 🎮 Úkoly ve hře

1. **Nakrm křečka** 🍎 - Drag & drop správného jídla do misky
2. **Najdi křečka v hoblinách** ✋ - Stírací los
3. **Poskládej křečka** 🧩 - Puzzle s 6 dílky (drag & drop)
4. **Chyť semínka** 🌻 - Chytání 10 padajících semínek do misky

Po dokončení všech úkolů se objeví video s křečkem!

- `index.html` - Hlavní HTML soubor
